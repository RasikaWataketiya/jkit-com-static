Consultant –FiCo/MM/SD/PP/PM

  • Extensive domain knowledge along with SAP configuration skills for at least 3 years.
  • At least one full life cycle SAP implementation project
  • Domain knowledge across multiple platforms, technologies, industries and functions would be an added advantage

Consultant - Basis

  • 3 years’ experience in SAP BASIS Administration, UNIX, or Windows operating systems as a SAP server platform
  • Experience in Oracle, Sybase and HANA databases and troubleshooting system SAP system performance problems

Consultant –ABAP/HANA

  • 3 years’ experience with Strong Conceptual knowledge in SAP ABAP development and implementation experience
  • ABAP workbench including: List, interactive, & ALV reporting; BAPIs BADIs and BDC scripts; SMART Forms, SAP Script; Dialog Programming; User-Exits & Implicit and explicit enhancements; Performance Tuning; ABAP Objects; Analyzing ABAP logs/dumps; Researching SAP Notes, OOPS, and Webdynpro
  • Strong SQL knowledge
  • Good hands on skills in RICEF objects
  • UI5 / SAP Fiori developements for mobile applications Optional

Senior Consultant – FiCo/MM/SD/PP/PM

  • Extensive domain knowledge along with SAP configuration skills for at least 5 years.
  • At least two full life cycle SAP implementation projects
  • Domain knowledge across multiple platforms, technologies, industries and functions would be an added advantage

Senior Consultant -Basis

  • 5 years of hands on experience in installation, upgrade and support of SAP BASIS NetWeaver environments.
  • Designed server architecture and performed sizing to define hardware required for the customer.
  • Experience in developing SAP data archive solutions, SAP HANA Enterprise Cloud installation, migration, support.
  • Knowledge in SAP HANA database and all well-known DBs like SYBASE, ORACLE and DB2.
  • Hands on experience supporting several SAP products like HANA database ECC, Business Warehouse (BW), Test Data Migration Server (TDM) , SAP Data Migration Option (DMO), SAP sFin, SAP sLog.
  • Minimum one HANA implementation or HANA migration (SOH / S4 HANA )

Senior Consultant ABAP/HANA

  • 5 years’ experience with Strong Conceptual knowledge in SAP ABAP development and implementation experience
  • ABAP workbench including: List, interactive, & ALV reporting; BAPIs BADIs and BDC scripts; SMART Forms, SAP Script; Dialog Programming; User-Exits & Implicit and explicit enhancements; Performance Tuning; ABAP Objects; Analyzing ABAP logs/dumps; Researching SAP Notes, OOPS, and Webdynpro,Idocs,Workflows
  • Strong SQL knowledge
  • Good hands on skills in RICEF objects
